## Install MongoDB for Windows
[Full docs for Window](https://docs.mongodb.com/manual/tutorial/install-mongodb-on-windows/)

Simple steps:
1- [Download and Install MongoDB](https://www.mongodb.com/download-center#community)

2- MongoDB requires a data directory to store all data. MongoDBâ€™s default data directory path is the absolute path \data\db on the drive from which you start MongoDB. Create this folder by running the following command in a Command Prompt:
```
    md \data\db
```

3- Start MongoDB
```
"C:\Program Files\MongoDB\Server\3.4\bin\mongod.exe"
```
